# Customer Segmentation using RFM Analysis

A Power BI dashboard that segments customers based on their purchase 
behavior using RFM analysis (Recency, Frequency, Monetary).

## What this project does

- Cleans and prepares customer order data
- Calculates RFM metrics for each customer
- Shows behavior patterns for each segment
- Suggests marketing actions for each segment
- Brings everything together in an interactive dashboard

## Tools used

- Power BI Desktop
- Excel – data source

## Dataset
AdventureWorksDW
RFM table
Customer RFM segments


## How RFM was calculated

- Recency: days since last purchase
- Frequency: number of orders
- Monetary: total amount spent
- Each metric scored 1–5 and combined into a 3-digit RFM score
- Scores mapped to 11 customer segments

## Dashboard pages

1. **Customer Demographics** – sales by gender, age, location
2. **Segmentation** – customer count and sales by segment, RFM breakdown
3. **Customer Details** – customer-level table
4. **Marketing Actions** – recommended action for each segment
